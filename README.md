## How to use locally/本地使用方式

#### 初始化

*PNPM*: 根目录下执行`pnpm i`

*NPM*: 根目录下执行`npm i`

#### 构建

*PNPM*：根目录`pnpm run build`

*NPM*：根目录`npm run build`

#### 开发

*PNPM*：根目录`pnpm run start`

*NPM*：根目录`npm run start`

tips:

不需要服务器可以使用watch

*PNPM*：根目录`pnpm run watch`

*NPM*：根目录`npm run watch`

如果需要开启vconsole，可以在url后面添加`?log=1`get参数
